Instruções para a execução do banco de dados:

  Abra o PgAdmin
  Clique em Databases
  Crie uma nova Database
  Após isso, execute algum dos métodos abaixo:

  Método 1 - Dumpfile do Backup do banco:
	- Clique com o botão direito no banco criado e selecione a opção Backup
	- Selecione o arquivo Backup-banco.sql, neste diretório
	- Clique em Backup

  Método 2 - Executar os códigos de criação manualmente:
	- Clique com o botão direito no banco criado e selecione a opção "Query Tool"
	- Copie o conteúdo do arquivo code.sql, neste diretório, e cole no Query Tool
	- Execute a query

As 10 consultas elaboradas, além de estarem no relatório com as suas avaliações, também
estão disponíveis no arquivo Consultas-SQL.txt, com os seus enunciados em forma de comentários,
bastando copiá-las e executá-las no banco, a fim de verificar e validar os seus resultados.
